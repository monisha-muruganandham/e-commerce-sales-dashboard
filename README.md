# E-Commerce Sales Analysis Dashboard (Excel)

## 📊 Project Overview
This project focuses on analyzing e-commerce sales data using Microsoft Excel to identify business trends and performance metrics.

## 🧹 Data Cleaning
* **Removed duplicate records** to ensure data accuracy.
* **Handled missing values** and standardized categorical values (e.g., payment methods).
* **Validated discount values** to ensure financial integrity.

## 📈 Analysis Performed
* **Calculated final revenue** after discounts using custom formulas.
* **Created Pivot Tables** for category, city, and product-level analysis.
* **Identified top-performing segments** to provide actionable insights.

## 📊 Dashboard Features
* **KPI Metrics:** Tracking Total Revenue, Top Category, Top City, and Top Payment Method.
* **Visual Reports:** Revenue by Category and City distribution.
* **Payment Method Analysis:** Visual breakdown of customer preferences.
* **Product Insights:** Detailed look at sales by specific items.

## 🔍 Key Insights
* **Electronics** generates the highest revenue.
* **Chennai** is the top-performing city in terms of sales volume.
* **UPI** is the most preferred payment method among customers.

## 🛠 Tools Used
* **Microsoft Excel:** (Pivot Tables, Charts, Data Cleaning, Protection).
* **GitHub:** Portfolio Management.
